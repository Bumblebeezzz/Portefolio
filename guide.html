Finding a job or clients — using this portfolio

No job is “guaranteed.” What improves your odds: concrete proof (live sites), warm network outreach, a clear niche, and a steady volume of targeted pitches.

1. What this portfolio shows

  • Lead Gen Tax (AU) — credibility for SMBs, tax, and Australian lead generation.
  • Tradeasy, Earthstralia, Sacred Quantum Alignment — live sites and web delivery.
  • Sacred Arithmetic — intellectual content and education.

2. Fast actions (recommended order)

  • Warm network — 20–30 messages to former contacts (“these projects are live; I’m looking for automation / web work”).
  • LinkedIn — headline such as “AI & automation · lead gen · SMB sites” + link to this hosted portfolio.
  • Upwork / Malt / marketplaces — apply within 15–30 minutes of new “automation / n8n / lead gen” job posts.
  • Targeted cold outreach — pick one niche (e.g. agencies, solar installers) + a short message with one link to a case study or this portfolio.

3. Host this portfolio on a public URL

  • Deploy this repository to Netlify, Vercel, or GitHub Pages.
  • Local preview: python -m http.server 8080 in the project folder (testing only).
  • With Flask: pip install -r requirements.txt then flask run (see README).
